# Asset files

Place the three files generated by `tools/gen_assets` here:

| File            | Contents                                              |
|-----------------|-------------------------------------------------------|
| `license.bin`   | Argon2id-encrypted licence (Ed25519-signed)           |
| `firmware.bin`  | AES-256-GCM encrypted firmware bytecode               |
| `codesign.bin`  | Ed25519 public key + signature over both asset hashes |

Generate with (from the library root):

```bash
CODESIGN_PRIVATE_KEY=<your_64_byte_hex_key> \
    cargo run --manifest-path tools/gen_assets/Cargo.toml -- \
    --package-id com.example.securevm.demo \
    --installer required:com.android.vending \
    --out android-app/app/src/main/assets
```

These files are binary and should NOT be committed to version control if
they contain a production signing key or licence secret.  Add them to
`.gitignore` and distribute them through your CI/CD secrets pipeline.
